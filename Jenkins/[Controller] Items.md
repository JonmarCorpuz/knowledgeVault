# Jenkins Item Overview

An item in Jenkins refers to an entity that executes specific tasks within the CI/CD pipeline

<br>

# Jenkins Items

## Freestyle Project

A freestyle project allows users to configure various build steps through the UI without needing a pipeline script

<br>

## Multi-Configuration (Matrix) Project

A multi-configuration/matrix project allows users to run a same build across different environments, configurations, or parameter combinations in parallel

* Useful for testing software in multiple versions and operating systems

<br>

## Pipeline

A pipeline is a Jenkins job that's defined by code using either declarative or scripted syntax

* The whole CI/CD pipeline is defined as code and versioned with the application source code for better maintainability and lifecycle management

<br>

## Multi-Branch Pipeline

A multi-branch pipeline 

* Automatically manages branches in source control repositories 
* Automatically creates pipeline jobs for each branch
