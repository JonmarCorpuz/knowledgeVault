# Terraform HashiCorp Cloud Platform Workspaces Overview

Workspaces in Terraform HCP are logical containers that group infrastructure resources, configurations, state files, and variables for Terraform management

<br>

# Workflow Types

## Version Control

Version Control workflows connect workspaces to a version control system repository (*GitHub*, *GitLab*)
