# Virtual Router Overview

A virtual router in a Palo Alto firewall is a logical routing instance that handles L3 routing for the interfaces assigned to it

* Each virtual router maintains its own RIB and FIB for the firewall

<br>

# Routing Information Base

The RIB of a virtual firewall refers to the full collection of routes that the device knows 

<br>

# Forwarding Information Base

The FIB of a virtual router refers to the optimized subset of routes that are actually used to forward packets
