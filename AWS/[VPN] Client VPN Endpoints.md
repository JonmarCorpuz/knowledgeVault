# Client VPN Endpoint Overview

Client VPN endpoints are managed VPN server resources that terminates TLS-based VPN sessions from individual clients 

<br>

# Required Client VPN Endpoint Components 

## Client IPv4 CIDR

The client IPv4 CIDR is the IPv4 address range from which AWS assigns tunnel IP addresses to connected VPN clients

<br>

## Server Certificate ARN

A server certificate ARN is the AEN that uniquely identifies a specific TLS server certificate stored in AWS Certificate Manager

<br>
